# Build a web page for a git repository with github pages

[https://github.com/pedroj/effectiveness_pckg](https://github.com/pedroj/effectiveness_pckg)

I built a GitHub Pages-ready static webpage for your repository and shared the HTML file above. The repository already contains the core package materials needed to support such a site, including `DESCRIPTION`, `NAMESPACE`, `R/`, `man/`, `README.md`, images, and an existing rendered `effectiveness.html` file, while the README describes `effect.lndscp` as an R package for plotting effectiveness landscapes of mutualisms with examples based on *Cecropia glaziovii* and *Prunus mahaleb* datasets.

## What’s included

The page is a single responsive HTML file with a polished landing-page layout, dark mode toggle, package overview, usage example, repository structure section, deployment guidance for GitHub Pages, and citation block based on the repository README and contents.

## How to publish

To publish it with GitHub Pages, place the shared file in your repository as `docs/index.html`, then enable GitHub Pages in the repository settings using the `docs/` folder on the default branch. That fits this repository well because it is already a standard package repository with documentation assets and an HTML artifact.
