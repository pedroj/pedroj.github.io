Effectiveness landscapes for mutualistic interactions.

effect.lndscp is an R package for plotting the effectiveness landscape of mutualisms, combining quantity and quality components in a two-dimensional representation with isolines of equal effectiveness.

It is built for seed dispersal and pollination studies, with included examples based on Cecropia glaziovii and Prunus mahaleb frugivore datasets.

What the package does

The package plots effectiveness landscapes as two-dimensional combinations of quantity and quality components of mutualistic services, with elevational contours representing isoclines of effectiveness.

Designed for mutualism effectiveness

The package targets multiplicative effectiveness components for seed dispersal and pollination, making it useful for visualizing how service quantity and quality combine into overall effectiveness.

Two included datasets

Examples in the README use datasets for Cecropia glaziovii and Prunus mahaleb frugivores.

Manual citation ready

The repository includes a package citation entry and DOI-based Zenodo reference for scholarly use.

GitHub Pages friendly

The repository already contains rendered HTML and supporting images, so a project page can point users directly to documentation and examples.

Typical workflow

  • Install the package from GitHub with devtools::install_github().
  • Load the package with library(effect.lndscp).
  • Use included datasets and effectiveness_plot() to visualize quantity–quality combinations.
  • Customize axis labels to match the ecological process and measurement units in your study.
# install from GitHub
devtools::install_github("pedroj/effectiveness_pckg")

# load package
library(effect.lndscp)

data(cecropia)
effectiveness_plot(
  cecropia$totvis,
  cecropia$totbic,
  myxlab = "No. visits/10h",
  myylab = "Effectiveness/vis (No. fruits handled)"
)
